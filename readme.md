# gitignore-cli

>

## Install

```
$ npm install --global gitignore-cli
```

```
$ gitignore-cli --help

  Usage:
    gitignore-cli [params]

  Examples:
    $ gitignore-cli java
	Creates .gitignore file for java
    $ gitignore-cli java node    
	Creates .gitignore file for java and node

```

#### params

Type: `array`

Example: java node c
